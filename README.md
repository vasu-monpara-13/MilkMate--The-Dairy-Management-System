# 📌 Project Overview

MilkMate is a **full-stack dairy management platform** designed to digitize milk subscriptions, delivery tracking, billing, and payments for dairy businesses.

The platform enables customers to subscribe to milk products, track deliveries, view monthly bills, and make online payments.

It also includes a **powerful admin dashboard** for monitoring dairy operations.

# 💻 Local Development

Clone repository

git clone https://github.com/yourusername/milkmate.git

Enter project

cd milkmate

Install dependencies

npm install

Setup environment variables

cp .env.example .env

Start development server

npm run dev

App runs at:

http://localhost:5173

---

# 🔐 Environment Variables

Example `.env.example`

VITE_SUPABASE_PROJECT_ID=your_project_id
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_PUBLISHABLE_KEY=your_public_key
VITE_RAZORPAY_KEY_ID=your_razorpay_key
